# Жобаның папкалар құрылымы (Project Structure)

```text
/
├── backend/                # Бэкенд (Python/FastAPI)
│   ├── audio/              # Ағылшын тілі үшін аудио файлдар (.mp3)
│   ├── data/               # Статикалық деректер (JSON)
│   ├── data_sources/       # Сұрақтардың бастапқы файлдары
│   ├── scripts/            # Мәліметтерді базаға салу (seed) және көмекші скрипттер
│   ├── tests/              # Автоматтандырылған тесттер
│   ├── auth.py             # Авторизация және JWT логикасы
│   ├── auth_routes.py      # Auth API эндпоинттары
│   ├── main.py             # Негізгі API сервері (FastAPI)
│   ├── models.py           # Деректер базасының модельдері
│   └── database.py         # DB қосылу баптаулары
├── components/             # React компоненттері (Frontend)
│   ├── AuthScreen.tsx      # Тіркелу және парольді қалпына келтіру
│   ├── HistoryScreen.tsx   # Тест тарихы экраны
│   ├── WelcomeScreen.tsx   # Басты бет
│   ├── TestScreen.tsx      # Тестілеу процесі
│   └── ResultScreen.tsx    # Нәтижелерді талдау
├── docs/                   # Техникалық құжаттамалар
├── public/                 # Статикалық активтер (суреттер, иконкалар)
├── resources/              # Оқу материалдары (PDF, MD)
├── services/               # API-мен жұмыс істеу сервистері
├── App.tsx                 # Негізгі React қолданбасы
├── types.ts                # TypeScript интерфейстері
└── start.py                # Жобаны бірден іске қосу скрипті
```

## Папкалардың қызметі
- **backend/**: Барлық серверлік логика мен деректер базасы.
- **components/**: Пайдаланушы интерфейсінің модульдері.
- **services/**: Бэкендпен байланысты реттейтін JavaScript функциялары.
- **resources/**: Пайдаланушыларға немесе әзірлеушілерге арналған анықтамалық ақпарат.
