# Жүйе құрылымы және Логика (System Architecture & Logic)

MagisCore — заманауи Single Page Application (SPA)
 архитектурасына негізделген веб-платформа.

## 🏗 Архитектуралық деңгейлер

### 1. Client Side (Frontend)
- **Технологиялар:** React 18, TypeScript, Vite.
- **Стиль:** Tailwind CSS + Framer Motion (анимациялар).
- **Мемлекетті басқару:** React Hooks (useState, useEffect) және Context API.
- **Интеграция:** REST API арқылы Бэкендпен байланысады.

### 2. Server Side (Backend)
- **Фреймворк:** FastAPI (Python).
- **ORM:** SQLAlchemy.
- **Қауіпсіздік:** JWT (JSON Web Tokens) авторизациясы.
- **Аутентификация:** PBKDF2-SHA256 хеширлеу алгоритмі.

### 3. Database (Storage)
- **SQLite:** Жеңіл және жылдам дерекқор.
- **JSON Serialization:** Тест нәтижелерін икемді сақтау үшін қолданылады.

## 🧠 Негізгі бизнес-логика

### Тест генерациялау логикасы (KT Форматы)
Ағылшын тілі пәні үшін жүйе **ҚР Магистратура (КТ)** стандартын сақтайды:
1. **Тыңдалым:** 16 сұрақ (2 аудио мәтін).
2. **Грамматика:** 18 сұрақ.
3. **Оқылым:** 16 сұрақ (2 мәтін).
Жүйе базадан сұрақтарды кездейсоқ (random) таңдайды, бірақ тақырыптық теңгерімді сақтайды.

### Нәтижені есептеу алгоритмі
1. **Single Choice:** 1 дұрыс жауап = 1 ұпай.
2. **Multiple Choice:** Толық дұрыс жауап үшін 2 ұпай (бейіндік пәндер үшін).
3. **Persistance:** Есептеуден кейін нәтиже бірден `test_results` кестесіне сақталады.

### Парольді қалпына келтіру (OTP Flow)
1. Пайдаланушы Email жазады.
2. Жүйе 6-таңбалы код генерациялайды.
3. Код Gmail SMTP арқылы жіберіледі.
4. 5 минуттық жарамдылық уақыты тексеріледі.
5. Код сәйкес келсе, пайдаланушыға парольді өзгертуге рұқсат беріледі.
