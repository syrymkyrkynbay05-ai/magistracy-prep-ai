# MagisCore - Жоба Статистикасы

Бұл құжатта платформадағы мәліметтердің қазіргі жай-күйі мен сандық көрсеткіштері берілген.

## 📊 Сұрақтар статистикасы
Платформада барлығы **800** сапалы сұрақ жинақталған:

| Пән | Сұрақ саны | Сипаттама |
| :--- | :---: | :--- |
| **English (Ағылшын тілі)** | 418 | Listening (аудио), Grammar, Reading |
| **TGO (Логика)** | 100 | Сыни және аналитикалық ойлау |
| **Algorithms (Алгоритмдер)** | 142 | C++, Мәліметтер құрылымы, Графтар, Big O |
| **Databases (SQL)** | 140 | ER-модельдеу, SQL, ACID, Нормализация |
| **Барлығы** | **800** | |

## 👥 Пайдаланушылар және Нәтижелер
- **Пайдаланушылар жүйесі:** Толыққанды тіркелу, кіру және парольді OTP арқылы қалпына келтіру функциялары іске қосылған.
- **Тест тарихы:** Әрбір аяқталған тест нәтижесі деректер базасында сақталады (ұпай, дұрыс/қате жауаптар, уақыты).

## 🛠 Техникалық мүмкіндіктер
- **Аудио базасы:** 17-ден астам кәсіби аудио файлдар (English Listening үшін).
- **AI Интеграциясы:** Сұрақтарды генерациялау және түсіндіру үшін Gemini AI prompt-тары дайындалған.
- **Масштабталу:** Жүйе кез келген уақытта жаңа сұрақтармен және пәндермен толықтырылуға дайын.
