## [1.0.0] - 2026-01-31

### ✨ Қосылды
- **Толық КТ симуляциясы** - 4 пән бойынша 500+ сұрақ
  - Шет тілі (Ағылшын) - 50 сұрақ
  - ОДАТ (Логика) - 30 сұрақ
  - Алгоритмдер - 30 сұрақ
  - Дерекқорлар (SQL) - 20 сұрақ
- **Маршрутизация** - React Router арқылы URL-ге негізделген навигация
  - `/home` - Басты бет
  - `/program/:subjectId` - Оқу бағдарламасы
  - `/test/:subjectId/q/:qIndex` - Тест тапсыру
  - `/result` - Нәтижелер
- **Анти-чит жүйесі**
  - Мәтінді көшіруді бұғаттау
  - Оң жақ батырманы өшіру
  - Аудармашыны болдырмау
  - Бетті жаңартудан қорғау
- **Деңгейлер жүйесі** - easy → medium → hard сұрыптау
- **Тақырыптық статистика** - Әлсіз тақырыптарды анықтау
- **Сертификат** - 60%+ нәтижеге цифрлық сертификат жүктеу
- **Оқу бағдарламасы** - Markdown форматында силлабус

### 🔧 Техникалық
- FastAPI backend с SQLite
- React 19 + TypeScript frontend
- Vite build tool
- Railway деплой конфигурациясы
